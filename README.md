# Task 3: Introduction to JavaScript

This project is part of **Level 1 (Basic)** web development tasks and focuses on introducing **JavaScript fundamentals** by adding interactivity to a static webpage.

The application demonstrates how JavaScript can be used to manipulate the DOM, handle user events, and validate form inputs in a clean and structured way.

---

## Features

- Dropdown interaction using JavaScript
- Modal popup with open and close functionality
- Client-side form validation
- Event handling with addEventListener
- Clean and responsive UI
- Separation of HTML, CSS, and JavaScript files
